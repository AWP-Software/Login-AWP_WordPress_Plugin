== Description ==

This plugin improves the login look for the website and defaults to the page logo

* Add a background image with transitions
* Put the main logo of the website
* Responsive
* Just activating it already adds personality to the login

== Frequently Asked Questions ==

= Can I change the background image? =

Yes, you can do it from directly change the image file located in `public\img\slider.jpg`, remember that it is expected to have the same name.

= What if I don't have a logo? =

The WordPress logo will be displayed by default.

= I changed the background image but it takes time to load =

Try reducing the size of the image or restoring the plugin to its initial version.
